# Android UI需求文档

## 1. 页面概述
- **页面名称**: 火星旅行预订界面
- **功能描述**: 用户可以在此页面选择从地球（DCA）到火星（MARS）的旅行选项，包括单程/往返、乘客数量，并最终确认出发。
- **所属模块**: 航天旅行服务

## 2. 组件列表

### 2.1 顶部导航栏
| 组件             | 类型  | 属性  | 说明          |
| -------------- | --- | --- | ----------- |
| Space Stations | Tab | 文本  | 左侧标签，当前未选中  |
| Flights        | Tab | 文本  | 中间标签，当前选中状态 |
| Rovers         | Tab | 文本  | 右侧标签，当前未选中  |

### 2.2 行程选择区域
| 组件 | 类型 | 属性 | 说明 |
|------|------|------|------|
| DCA | TextView | 文本, 背景色: #006400 | 地球站点标识，绿色背景 |
| MARS | TextView | 文本, 背景色: #006400 | 火星站点标识，绿色背景 |
| 双向箭头 | ImageView | 图标 | 连接DCA和MARS的双向箭头图标 |
| One Way | Switch | 开关控件, 背景色: #C95B00 | 单程选项开关，橙色背景 |
| 1 Traveller | TextView | 文本, 背景色: #C95B00 | 乘客数量显示，橙色背景 |

### 2.3 宇宙飞船动画
| 组件 | 类型 | 属性 | 说明 |
|------|------|------|------|
| 飞船图标 | ImageView | 图标 | 小型火箭图标，位于左侧 |
| 火星轨道 | ImageView | 图标 | 火星轨道图形，包含中心红色圆圈和外围轨道 |

### 2.4 出发按钮
| 组件 | 类型 | 属性 | 说明 |
|------|------|------|------|
| DEPART | Button | 按钮, 背景色: #006400, 文本颜色: #FFFFFF | 底部绿色大按钮，用于确认出发 |

## 3. 交互流程

### 3.1 页面加载
- 显示顶部导航栏，Flights标签高亮
- 显示行程选择区域，初始状态为One Way开启，1 Traveller
- 显示宇宙飞船动画
- 显示DEPART按钮

### 3.2 用户交互
1. **切换行程类型**:
   - 点击One Way开关，可在单程和往返之间切换
   - 切换时应有平滑的动画效果

2. **修改乘客数量**:
   - 点击1 Traveller区域，弹出乘客数量选择器
   - 用户可选择1-5名乘客
   - 选择后更新显示文本

3. **确认出发**:
   - 点击DEPART按钮
   - 触发出发动画，飞船从地球飞向火星
   - 跳转到行程确认页面

## 4. 设计规范

### 4.1 颜色方案
- **主色调**: #006400 (深绿色)
- **辅助色**: #C95B00 (橙色)
- **文字颜色**: #FFFFFF (白色)
- **背景色**: #FFFFFF (白色)

### 4.2 字体规范
- **标题字体**: Roboto Bold, 18sp
- **正文字体**: Roboto Regular, 14sp
- **按钮字体**: Roboto Medium, 16sp

### 4.3 尺寸规范
- **顶部导航栏高度**: 56dp
- **行程选择区域高度**: 100dp
- **DEPART按钮高度**: 56dp
- **飞船图标大小**: 24dp × 24dp
- **火星轨道图标大小**: 100dp × 100dp

## 5. 备注
- 所有UI元素需符合Material Design规范
- 动画效果需流畅自然
- 确保在不同屏幕尺寸上的适配性
- 需要实现无障碍访问支持
